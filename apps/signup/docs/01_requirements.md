# 1단계 · 요구사항 분석

[기능 명세](../README.md#기능-명세-v10) 를 5가지 기준으로 분해한다.

## 5가지 분석 기준
1. 사용자
2. 주요 기능
3. 상태 변화
4. 예외 상황
5. 외부 시스템 또는 정책 조건

## 체크리스트
- [ ] 입력 필드별 검증 규칙(길이 / 형식 / 문자 조합) 수치까지 항목화
- [ ] 검증 트리거 시점(실시간 vs 블러 vs 제출) 구분
- [ ] 약관 필수 / 선택 분리, 연령 확인 별도
- [ ] 이메일 상태 분기(미가입 / 가입 완료 / 인증 대기) 명시
- [ ] 시간 조건(24시간 인증 유효, 자동 폐기) 항목화
- [ ] 보안 / 법적 정책(평문 금지, 사용자 열거 방지, 연령 제한) 별도 항목
