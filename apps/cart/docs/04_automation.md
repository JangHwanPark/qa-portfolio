# 4단계 · 자동화 분류 + 매뉴얼 분리

[03_testcases.md](03_testcases.md) 의 TC 를 **매뉴얼 / 자동 / 부분자동 / 제외** 로 분류한다.

## 분류 정의
- **매뉴얼**: 사람이 봐야 가치 있는 검증 (시각, UX, 자연스러움)
- **자동**: 매 PR Smoke 또는 nightly Regression 에서 자동 실행
- **부분 자동**: 사전 상태 세팅(API / 페이지 reload) 필요
- **자동 제외**: 외부 의존, 브라우저별 차이가 커서 자동화 비용 대비 가치 낮음

## 작성 조건
- 각 TC 마다 분류 + 근거 명시
- 셀렉터 전략 / 대기 전략 별도 섹션 작성
- 코드 위치 파일 경로 명시
- 매뉴얼 검증 절차도 별도 표로 정리 (실제로 진행할 수 있게)
